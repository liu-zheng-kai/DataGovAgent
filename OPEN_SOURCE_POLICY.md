# Open Source Policy

This repository is licensed under `AGPL-3.0-or-later`.

That means:

- If you copy, modify, distribute, or deploy a modified version of this project for users over a network, you must provide the complete corresponding source code of your version under the same AGPL terms.
- You must preserve copyright and license notices.
- You may not convert this project into a closed-source derivative when AGPL obligations apply.

## Public Publication Expectation

The binding legal terms are defined by [LICENSE](./LICENSE).

In addition to those license terms, this repository states the following public reuse expectation:

- If an individual or organization reuses or modifies this project for external sharing or public service deployment, the modified source code should be published in a public Git repository.
- GitHub is the preferred publication destination.
- The published repository should retain attribution to `DataGovAgent` and link back to the original repository when reasonably possible.

## Note

AGPL is the core enforceable open source license used by this repository.
The GitHub publication requirement is stated as a repository policy and reuse expectation.
